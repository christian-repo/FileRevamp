## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-01 | Tampering | FailureLogger — log file path derived from directoryPath | mitigate | Path.Combine(directoryPath, "rename-failures.log") — log filename is a hardcoded constant; no user-controlled path segments in the log filename; directoryPath already Path.GetFullPath resolved (T-01-02) |
| T-04-03 | Information Disclosure | rename-failures.log reveals filenames and failure reasons | accept | RPRT-03 requirement explicitly requires this; developer CLI tool, no multi-user surface; user's own files |
| T-04-SC | Tampering | npm/pip/cargo installs | accept | No new packages; BCL File.AppendAllText only |
</threat_model>
